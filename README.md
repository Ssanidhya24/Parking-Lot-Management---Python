# Parking-Lot-Management---Python

This project implements a simple Parking Lot Management System using Python.  
The application supports:

- Three types of parking slots: SMALL, LARGE, OVERSIZE  
- Vehicle parking
- Vehicle exit
- Displaying the status of all slots

It is a command-line based application for now

- Assign the first suitable open slot to a vehicle based on its size 
- It also Prevent double-parking of the same vehicle  
- Track which vehicle occupies which slot using a dictionary style lookup  
- Display a clean table of parking lot status  
- Includes data for testing
